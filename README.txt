#🐾 Plataforma de Adoção de Animais

Projeto desenvolvido com o objetivo de facilitar o processo de adoção de animais, conectando usuários com ONGs que cadastram e gerenciam os animais disponíveis diretamente pela plataforma.

 💻 Funcionalidades

- Cadastro e login de usuários e ONGs  
- Publicação de animais com informações detalhadas  
- Página de produtos para os animais 
- Filtros por espécie, porte, localização e outros  
- Interface responsiva e de fácil navegação  

🛠️ Tecnologias utilizadas

- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** PHP  
- **Banco de dados:** MySQL  

📌 Objetivo

Ajudar a promover a adoção responsável de animais e dar visibilidade ao trabalho das ONGs, oferecendo uma plataforma digital simples, direta e funcional.
